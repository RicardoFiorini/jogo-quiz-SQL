# Jogo Quiz SQL

Um jogo interativo de perguntas e respostas projetado para testar conhecimentos em **SQL** e banco de dados. Este projeto foi desenvolvido para consolidar conceitos de modelagem de dados e estruturação de consultas de forma prática e divertida.

## Funcionalidades

- Sistema de perguntas de múltipla escolha abordando comandos DDL, DML e DQL.
- Cálculo de pontuação em tempo real ao final de cada rodada.
- Feedback imediato de acertos e erros para facilitar a memorização.
